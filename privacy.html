Privacy Policy

Last updated: March 27, 2026

Porta Proxy ("the Extension") is a Chrome browser extension that manages proxy server settings and enables selective domain routing. This Privacy Policy describes how the Extension handles user data.

1. Data Collection

Porta Proxy does NOT collect, transmit, or share any personal data.

The Extension stores only the following configuration data locally on your device using Chrome's chrome.storage.local API:

  • Proxy server address (IP/host) and port
  • Proxy authentication credentials (login and password)
  • Routing mode preference (all sites or selected sites)
  • List of selected domain names for routing
  • Activity log entries (up to 30 most recent events)

All of this data is stored exclusively on your local device and is never transmitted to any external server, analytics service, or third party.

2. Permissions Justification

The Extension requires the following Chrome permissions:

  • proxy — Required to configure Chrome's proxy settings (fixed server mode and PAC scripts for selective routing).
  • storage — Required to save your proxy configuration locally so settings persist between browser sessions.
  • webRequest — Required to monitor network activity for the activity log feature and to detect proxy connection errors.
  • webRequestAuthProvider — Required to automatically handle proxy authentication challenges so users don't see browser login popups.
  • host_permissions: <all_urls> — Required because the proxy manager must be able to intercept and route ALL web traffic through the configured proxy server. The Extension cannot know in advance which URLs the user will visit, therefore access to all URLs is necessary for core functionality.

3. Third-Party Services

Porta Proxy does not integrate with any third-party analytics, tracking, or advertising services. The Extension does not make any network requests of its own — it only configures Chrome's built-in proxy mechanism.

4. Data Security

Your proxy credentials and configuration are stored using Chrome's built-in storage API, which is sandboxed to this Extension only. No other extensions or websites can access this data.

5. Data Deletion

You can delete all stored data at any time by:

  • Clearing the activity log via the "ОЧИСТИТЬ" button in the Extension
  • Uninstalling the Extension (removes all stored data automatically)

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected by updating the "Last updated" date at the top of this page.

7. Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository or contact us at the email address listed on the Chrome Web Store listing.
